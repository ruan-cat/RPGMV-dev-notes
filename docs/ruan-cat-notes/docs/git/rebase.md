# rebase 变基命令

## 用 rebase 代替 merge

阅读了一些文章。称最好用 git rebase 功能来做代码合并。用 rebase 变基功能代替单纯的分支合并。这个功能能更好的让 git 提交记录简洁清晰，呈现单线的提交记录。

良好清晰的 git 提交记录能够有效地记录开源项目的历史提交内容。
