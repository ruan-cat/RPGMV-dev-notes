# volar vscode 针对 vue3 生态链的语言服务器插件

## volar 对 vue2.7 的类型支持不完善

我仍旧有很多项目停留在 vue2.7 版本内，需要使用合适的配置，让 volar 给予 vue2.7 项目最好的类型推断支持。
