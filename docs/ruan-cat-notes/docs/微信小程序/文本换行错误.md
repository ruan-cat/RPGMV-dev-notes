# 文本换行错误

原生的微信小程序，其 `<text>` 标签有较大的问题。

## 具体现象

不同的代码换行效果，导致小程序的文本出现了不同的换行情况。代码格式化功能反而影响了页面的文本布局。

### 无误代码

```html
<view>
	报名
	<text class="registration-num">{{item.alumnInfoList.length}}</text>
</view>
```
