### 评价

感觉有点东西：

## 自动化文档生成的整体进度

截止目前，比较熟悉的文档生成框架是 jsdoc 和 typedoc。

按照数据源来划分，数据源对象可以是：

- .js 文件
- .ts 文件
- .vue 文件

按照注释格式划分，可以被划分为：

- jsdoc
- tsdoc
- 框架自有的格式

按照生成结果划分。可以被划分为：

- html
- md
