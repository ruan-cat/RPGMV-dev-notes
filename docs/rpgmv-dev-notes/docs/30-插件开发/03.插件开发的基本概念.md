# 插件开发的基本概念

## 概论

首先必须明确 mv 项目的定位：

mv 项目指的是，由`RPGMV`生成的基于`JavaScript ES5`版本的 PC 端/移动端 项目。

## 专有名词

这里罗列了一些本网站常用的专有名词。

### SIAKO.Mobi

指的是一个在台湾的 RPGMV 游戏开发界的开发者。

### 鳗驼螺
