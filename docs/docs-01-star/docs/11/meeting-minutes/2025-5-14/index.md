## git分支名称

命名规范很严格：

1. 分支名必须满足`f1-*`的前缀。不满足该glob语法匹配格式的分支名，无法在后续被筛选出来。影响分支整合工作。
2. 分支名不允许出现任何*中文、特殊字符*等。这些分支名是有意义的*变量名*。
3. 分支名不需要完全遵守[用户名](../2025-5-11/index.md#明确自己统一的用户名)规则。

与此同时，去腾讯文档内填写你们的分支名，便于后续统一整合代码。
